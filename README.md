## OthloHackathon

## How to Use
1. `git clone https://github.com/OthloTech/OthloHackathon.git`
2. `git checkout feature/2017`
3. `npm install`
4. `npm start`    // 変更を検知 & サーバ立ち上げ
5. `localhost:8080`にアクセス

## Requirements
- Python3
